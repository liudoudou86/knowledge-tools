## 技术细节

### 智能启动检测机制

```python
def wait_for_flask_ready(max_wait=10):
    """智能等待Flask服务就绪"""
    start_time = time.time()
    while time.time() - start_time < max_wait:
        try:
            response = requests.get('http://127.0.0.1:5000/api/health', timeout=1)
            if response.status_code == 200:
                return True
        except:
            time.sleep(0.5)
    return False
```
